# spendify

 Overview

Spendify is a modern Flutter-based personal finance application that helps users track expenses, visualize spending patterns, and manage their finances more intentionally.

Built with clean UI principles and dynamic state management, Spendify focuses on simplicity, usability, and data-driven insights.

This project demonstrates mobile development skills, UI/UX design, state management, and data visualization.

 Features

 Add, edit, and delete expenses

 Dynamic spending reports with charts

 Dark & Light mode toggle

 Custom Material 3 theme

 Real-time UI updates

 Responsive and user-friendly interface

 Reporting & Analytics

Spendify includes built-in data visualization to help users understand their spending trends.

Monthly expense tracking

Visual charts for spending analysis

Clean breakdown of financial activity

This demonstrates practical implementation of data visualization in Flutter.
